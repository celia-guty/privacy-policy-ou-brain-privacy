Política de Privacidade - BRAIN

Última atualização: 30 de janeiro de 2026

1. Introdução

A Cognitive Insights ("nós", "nosso" ou "empresa") opera o aplicativo BRAIN - Behavior Research & Applied Neuroscience ("Aplicativo"). Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informações pessoais quando você utiliza nosso Aplicativo.

Ao utilizar o Aplicativo BRAIN, você concorda com a coleta e uso de informações de acordo com esta política. Recomendamos a leitura completa deste documento antes de utilizar nossos serviços.

2. Informações que Coletamos

2.1 Dados Demográficos

Coletamos as seguintes informações fornecidas voluntariamente pelo participante:

Dados Coletados

2.2 Dados de Pesquisa

Durante a realização dos testes, coletamos:

•
Respostas às perguntas do questionário consciente (níveis de estresse, esgotamento emocional, prática de atividade física)

•
Tempo de reação nas tarefas de priming afetivo

•
Respostas comportamentais (classificação de estímulos como positivos ou negativos)

•
Dados de desempenho no teste (acertos, erros, timeouts)

2.3 Dados Técnicos

Podemos coletar automaticamente:

•
Tipo de dispositivo e sistema operacional

•
Data e hora de acesso

•
Identificadores únicos de sessão

3. Finalidade do Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

1.
Pesquisa Científica: Análise de padrões comportamentais e cognitivos relacionados ao bem-estar no ambiente corporativo.

2.
Relatórios Agregados: Geração de relatórios estatísticos para as empresas contratantes, sempre de forma anonimizada e agregada.

3.
Melhoria do Serviço: Aprimoramento contínuo da metodologia e experiência do usuário.

4.
Cruzamento Técnico: A identificação do participante é necessária apenas para possibilitar o cruzamento técnico entre diferentes métodos de pesquisa utilizados no estudo.

4. Compartilhamento de Dados

Não compartilhamos dados pessoais identificáveis com terceiros.

Os dados são tratados da seguinte forma:

•
Empresas Contratantes: Recebem apenas relatórios agregados e anonimizados, sem possibilidade de identificação individual dos participantes.

•
Pesquisadores: Podem ter acesso a dados anonimizados para fins de pesquisa científica.

•
Autoridades: Podemos divulgar informações quando exigido por lei ou ordem judicial.

5. Armazenamento e Segurança

5.1 Medidas de Segurança

Implementamos medidas técnicas e organizacionais para proteger seus dados:

•
Criptografia de dados em trânsito (HTTPS/TLS)

•
Armazenamento em servidores seguros

•
Controle de acesso restrito aos dados

•
Backups regulares

5.2 Período de Retenção

Os dados são armazenados pelo período necessário para cumprir as finalidades descritas nesta política, respeitando os prazos legais aplicáveis. Dados de pesquisa podem ser mantidos de forma anonimizada por tempo indeterminado para fins científicos.

7. Uso por Menores

O Aplicativo BRAIN é destinado exclusivamente a maiores de 18 anos. Não coletamos intencionalmente dados de menores de idade. Se tomarmos conhecimento de que coletamos dados de um menor, tomaremos medidas para excluí-los imediatamente.

8. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos sobre quaisquer alterações significativas através do Aplicativo ou por outros meios apropriados. Recomendamos revisar esta política regularmente.
